# react-udemy
React udemy course "Iniciando com React"

## Usage
- Clone or donload
- Enter application folder
```bash
$ cd react-udemy
```
- Install Gulp globally
```bash
$ npm install --global gulp
```
- Install packages
```bash
$ npm install
```
- Turn on application http server
```bash
$ npm server
```
- Build frontend
```bash
$ gulp
```
